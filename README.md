# Fruit Adventure

Um jogo de plataforma 2D clássico desenvolvido como projeto para o tutorial de Introdução ao Unity da disciplina de Jogos Digitais (2026.1).

Corra contra o tempo, colete todas as frutas espalhadas pelo cenário e desvie do inimigo patrulheiro e dos abismos perigosos para vencer!

**[Jogue a versão WebGL no Itch.io aqui!](https://iaravsp.itch.io/fruit-adventure)**

## Controles

O jogo possui suporte nativo para teclado e controle de Xbox.

**Teclado:**
* **Mover:** Setas direcionais (⬅️ ➡️) ou teclas A e D.
* **Pular:** Barra de Espaço.

**Controle (Xbox):**
* **Mover:** Analógico esquerdo ou D-Pad.
* **Pular:** Botão **Y**.
